INSTALLATION

BBOOTS ON EXISTING FORUMS

In order to integrate the Bootstrap framework with phpBB3 there are few core edits that are necessary to be installed. The core edits are in modx format so that you might use AutoMod to install it automatically without the need to manually edit each php file. The core edit file is within this documentation folder. (SiteSplat_BBCore.zip)

How to:
  1. Log in to your phpBB3 Admin Control Panel(ACP).
  2. Click the AutoMod tab in the top navigation bar. (If you have NOT yet installed the AutoMOD tool, you will need to download and install it from the phpBB website and install it. Download AutoMod.) - If you are new to phpBB, the tutorial at the following link will explain you how to install succesfully AutoMod: http://www.sitesplat.com/phpBB3/viewtopic.php?f=10&t=874
  3. Scroll down to the “Upload MOD” section and upload the "SiteSplat_BBCore.zip" file from the documentation folder.
  4. You may see a message confirming that the plugin has been uploaded. If so, click “Continue to AutoMOD.”
  5. Click the “Install” link to complete the installation.
  6. Upload the "BBOOTS" folder contained in: MAIN-THEME/Bootstrap3x/001-ElectricBlue/BBOOTS to your styles folder of your forum root (Where the index.php resides).
  7. From the Administration Control Panel (ACP) and within the "STYLES" tab choose BBOOTS (previously uploaded) and install it.
  8. Make BBOOTS the default style
  9. Done! Enjoy the style ;-)
FORUM MENU AND MOBILE MENU

FORUM MAIN MENU

To change edit/add links for the forum regular and mobile navigation open: BBOOTS/template/navigation.html and edit the links to your liking.

Adding menu sections

With notepad++ right click on the navigation.html and hit edit.

Find:

<nav class="mainnav"><!-- Main navigation block -->

That is the start of the desktop navigation. For a new section add:

<li class="nav-icon">
<a href="#">your main title <span class="has-sub"><i class="icon-home"></i></span></a>
<p>your subtitle</p>
</li>

To add sub-elements to a new section add:

<ul class="submenu">
<li><a href="{U_ELEMENTS}"><i class="icon-magic"></i> {L_BOOTSTRAP_ELEMENT}</a><span></span></li>
</ul>

Your full code block should look like this:

<li class="nav-icon">
<a href="{U_INDEX}">{L_HOME} <span class="has-sub"><i class="icon-home"></i></span></a>
<p>{L_MAIN_MAIN_STUFF}</p>
<ul class="submenu">
<li><a href="{U_ELEMENTS}"><i class="icon-magic"></i> {L_BOOTSTRAP_ELEMENT}</a><span></span></li>
</ul>
</li>

MOBILE MENU

For the Mobile navigation look for this line:

<nav class="mainnav-mob"> <!-- MOBILE Navigation block -->

Adding elements:

<li><a href="{U_PROFILE}"><i class="icon-dashboard"></i> {L_PROFILE}</a></li>

adding a full section with submenu:

<li class="item-with-ul">
<a href="#4"><i class="icon-search"></i> Submenu Group 1</a>
<ul class="collapse viewdetails">
<li><a href="#5">SubLink 1</a></li>
<li><a href="#6">SubLink 2</a></li>
<li><a href="#7">SubLink 3</a></li>
<li><a href="#8">SubLink 4</a></li>
</ul>
<a class="touch-button" href="javascript:void(0);" data-toggle="collapse" data-target=".viewdetails"><i class="navicon icon-sort-down"></i></a>
</li>

FORUM FONTS

FORUM FONT

You can change the font by adding and removing the through a simple link generated with google free font library

Choose your font from here: Google free font then in the overall_header.html find:

<!-- google free font used for the foum content -->

then add your font link below that line:

<link href='http://fonts.googleapis.com/css?family=Source+Sans+Pro:400,600' rel='stylesheet' type='text/css'>
<link href='http://fonts.googleapis.com/css?family=Roboto+Condensed:400,700' rel='stylesheet' type='text/css'>

Now you need to call the font family for the various elements. In this case for the menu font open: Basic.css located in the theme folder and around line: 1916 change the primary font family name, font size etc:

font-family: 'BebasNeueRegular','Arial','Helvetica',sans-serif;

Your code should look like this:

.mainnav > ul > li > a {
    color: #8C8C8C;
    display: block;
    font-family: 'BebasNeueRegular','Arial','Helvetica',sans-serif;
    font-size: 28px;
    font-style: normal;
    font-weight: 400;
    line-height: 20px;
    min-width: 89px;
    padding: 10px 5px 40px;
    text-decoration: none;
    text-transform: uppercase;
}

TWITTER

TWITTER ID FOR THE FORUM FOOTER

In order to get the Twitter feed from your own Twitter page you need to have a widget ID generated from your Twitter account and then add the ID to the forum footer.

This video shows how to generate a twitter ID from the widget tool: ahttp://www.youtube.com/watch?v=Ypsib-Nx4VQ

Now open the overall_footer.html and find this code:

391407906655965184

Replace this number: 391407906655965184 with your newly generated twitter ID:

twitterFetcher.fetch('391407906655965184', 'example1', 3, true);
SET YOUR OWN LOGO

LOGO

Open the overall_header.html located in the template folder, with a text editor like: Notepad++ and around line 146 find:

<a class="logo" href="{U_INDEX}" title=""><span>B</span>BOOTS</a>

Replace it with:

<a class="logo" href="{U_INDEX}" title=""><img alt="mylogo" src="{T_THEME_PATH}/images/mylogo.png"></a>

Now upload your logo in the theme/images/ folder.

In order to have it dsiplayed you also need to refresh your theme,forum and browser cache.

A) Purge the template and theme :

1) Navigate to "Styles" Tab.
2) Under "Style components " click on "Templates".
3) Under "Actions" click on "Refresh" for the BBOOTS style installed.
4) in the next screen, click on "Yes" to confirm
5) Repeat the steps for the "Themes" tab right under the "Templates" tab

B) Purge the cache :

1) Navigate to General Tab
2) look at Purge the cache
3) Click on "Run Now" button at right
4) in the next screen, click on "Yes" to confirm

C) Purge the browser cache :

1) Hit F5 on your browser or CTRL+F5

NON-ENGLISH LANGUAGE / TRANSLATIONS

LANGUAGES

You can add your own language pack to your forum by downloading and installing the official phpBB3 language packs here: Language Packs

Once the language pack is installed you need to translate the BBOOTS's own language file. This file gets installed automatically with the SiteSplat_BBCore.zip package.

The location is: language/en/mods/sitesplat.php. Open sitesplat.php file with notepad++, translate the content and save. Now you need to add the bboots.php file to your language pack (previosuly installed) directory. For example if you installed French, the location where to copy the sitesplat.php file will be in: root/language/fr/mods/

You can also check out the official support forum at: BBOOTS Language Packs and see if there is already a language file translated.
